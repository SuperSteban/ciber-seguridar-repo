PASO 1 Crear el entorno virtual

```sh
python -m venv .venv
```
PASO 2 Activar entorno

Unix

``` sh
source .venv/bin/activate
```
Windows, cmd
si usas powershell abrir como admin 

``` sh
.\.venv\Scripts\Activate
```

Instalar paquetes
```sh
python -m pip install -r requirements.txt

```

Salir de Entorno

```sh
deactivate
```
```sh
.\.venv\Scripts\deactivate
```
